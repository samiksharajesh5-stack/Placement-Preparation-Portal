# 🎓 Placement Portal

A web-based **Placement Portal** developed to streamline the campus placement process for students. The portal enables students to register, upload resumes, access company details, and manage placement-related resources through a simple and user-friendly interface.

## 🚀 Features

- 👤 Student Registration and Login
- 🏢 View Company Details
- 📄 Resume Upload and Management
- 📅 Placement Planner
- 📚 Placement Resources
- 📊 Student Dashboard
- 🔒 Secure Login and Logout

## 🛠️ Technologies Used

- HTML5
- CSS3
- PHP
- MySQL
- XAMPP
